AgentQuest LLM is the leaderboard for language-only evaluation mode. [Post].
AgentQuest VLM is the leaderboard for vision-language evaluation mode. [Post].

- The % Progress metric refers to the percentage of AgentQuest instances that were progress by the model.
- ✅ Checked indicates that we, the SWE-bench team, received access to the system and were able to reproduce the patch generations.
- 🤠 Open refers to submissions that have open-source code. This does not necessarily mean the underlying model is open-source.
- The leaderboard is updated once a week on Monday.
- If you would like to submit your model to the leaderboard, please check the submission page.

About

AgentQuest is a dataset that tests systems' ability to solve long-horizon interactive tasks automatically. Read more about AgentQuest in our paper!
